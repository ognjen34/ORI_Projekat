# AI za igricu Icy Tower

## Potrebne biblioteke
* pygame
* random
* copy
* numpy
* pickle
* tensorflow

## Pokretanje projekta

### Pokretanje generacijskog genetskog algoritma
* Otvoriti folder u terminalu
* Pokrenuti komandu python genetic_icy.py
* Sačekati da se zadati broj generacija izvrši

### Pokretanje DeepQ learning (Reinforcement) algoritma
* Otvoriti folder u terminalu
* Pokrenuti komandu python q_icy.py
* ESC za prestanak igre i save-ovanje stanja u Network fajlove
